## Dependencies
```
Python 3.12.4
```

## Virtual Environment
### Install pipenv

```
brew install pipenv [For MAC]
apt-get install pipenv
```

### Creating the pipenv virtual environment
```
pipenv --python 3.12
```

### Activating the virtual environment
```
pipenv shell
```

### Install pipenv in virtual environment

```
pip install pipenv
```

### Install Dependencies
```
pipenv install
```
